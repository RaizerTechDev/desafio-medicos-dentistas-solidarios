## 📄 Sobre o Projeto

Este projeto foi desenvolvido como parte de um desafio técnico do curso **Vai na Web**. O objetivo foi criar uma Landing Page institucional completa, responsiva e funcional utilizando **React**.

A grande complexidade deste desafio reside nas **restrições técnicas** impostas para testar o domínio profundo dos fundamentos do CSS, forçando a resolução de problemas de layout sem o uso de facilitadores modernos ou frameworks.

---

</br> 

## ⚠️ O Desafio e Regras

Para garantir a fixação dos conceitos de alinhamento e estrutura, o desafio impôs regras estritas. O layout teve que ser construído "na unha", testando a lógica do desenvolvedor.

### ✅ Requisitos Obrigatórios:
- Uso de **React**.
- Estruturação total feita com **Flexbox**.
- Responsividade completa (Mobile, Tablet e Desktop).
- Uso de unidades relativas (`%`, `vh`, `vw`, `rem`, `em`).
- Uso de Google Fonts (Fonte Inter).

### ❌ Estritamente PROIBIDO (Constraints):
Para elevar o nível de dificuldade, **não foi permitido** o uso de:
- 🚫 **CSS Grid**
- 🚫 **Position** (Absolute, Relative, Fixed, Sticky...)
- 🚫 **Frameworks CSS** (Bootstrap, Tailwind, Bulma...)
- 🚫 **Bibliotecas de UI** (Material UI, Chakra, AntDesign...)
- 🚫 **Plugins externos**

*Obs: Todos os alinhamentos, sobreposições visuais e estruturas de colunas foram resolvidos puramente com o fluxo padrão do DOM e propriedades de Flexbox.*

## 📦 Como Rodar o Projeto

Pré-requisitos: Node.js instalado.

1. **Clone o repositório:**
    ```bash
   git clone https://github.com/seu-usuario/nome-do-repo.git
   
    ```
2. **Acesse a pasta do projeto:**
    ```bash
   cd nome-do-repo
    ```
3. **Criar o projeto com vite:**
    ```bash
   npm create vite@latest
    ```
4. **Instale as dependências:**
    ```bash
   npm install
   npm install sass
   npm install react-router-dom
    ```
5. **Rode o servidor de desenvolvimento:**
    ```bash
   npm run dev
    ``` 
6. **Acesse no navegador:**
    ```bash
   O terminal exibirá o endereço local, geralmente: http://localhost:5173
    ```
